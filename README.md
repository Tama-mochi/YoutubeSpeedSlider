# 使用方法

- [ ] 任意の場所に**youtube-speed-slider**という名前のフォルダを作成
- [ ] **content.js/manifest.json/styles.css** を保存
- [ ] **chrome://extensions** を開く
- [ ] **開発者モード**をオンにする
- [ ] **展開して読み込み**をクリックして**youtube-speed-slider**フォルダを開く
- [ ] Youtubeを開いて動作確認
